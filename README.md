# Danganronpa V3 Treasure Hunter Monolith Solver

A static, client-side web solver for the Treasure Hunter Monolith minigame in Danganronpa V3: Killing Harmony.

## Live URL

TBD (GitHub Pages)

## Tech

- HTML
- CSS
- JavaScript
- No backend, no build step

## Usage

1. Open `index.html` in your browser, or host the repository with GitHub Pages.
2. Load a screenshot of your board.
3. Parse, calibrate if needed, and follow the generated solution steps.
